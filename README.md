# 🐧 Linux Essentials

¡Bienvenido a **Linux Essentials**! Este repositorio es una bitácora personal y una guía de referencia rápida diseñada para dominar el sistema operativo Linux desde la línea de comandos hasta la práctica avanzada.

Aquí encontrarás comandos esenciales, conceptos clave, configuración de entornos y ejercicios prácticos resueltos.

### 🗺️ Contenido del Repositorio

El proyecto se divide en las siguientes secciones (en constante actualización):

* **Comandos de Consola:** Glosario de herramientas organizadas por categorías (navegación, permisos, redes).
* **Datos y Conceptos de Linux:** Notas teóricas sobre la arquitectura del sistema, el sistema de archivos y el kernel.
* **Ejercicios Prácticos:** Retos, scripts de Bash y laboratorios para poner a prueba tus conocimientos a largo plazo.
* **Scripts Útiles:** Automatizaciones listas para usar en tu entorno diario.
